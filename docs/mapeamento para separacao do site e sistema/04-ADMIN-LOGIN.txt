Q1 — RESPONDIDA (doc oficial TanStack Start/Router)
----------------------------------------------------------------
SIM. head() funciona por rota e COMPOE com o __root.
- routeOptions.head retorna { title, meta, links, styles, scripts }.
- Router faz DEDUPE automatico de title e meta (vence a ocorrencia
  mais profunda; nested route sobrescreve parent).
- Tags carregam/descarregam conforme visibilidade da rota.
- Requer <HeadContent /> e <Scripts /> renderizados (JA TEMOS no __root).
Fonte: tanstack.com/router .../document-head-management
       tanstack.com/start .../guide/seo (exemplo em src/routes/index.tsx)

>> IMPLICACAO DIRETA: podemos mover SEO/OG/JSON-LD/fonts/styleHrefs
   do site PARA index.tsx via head() da rota '/'. O title/description
   da landing vai sobrescrever o neutro do __root por dedupe. CSS e
   scripts do site so carregam quando '/' esta ativa. EXATO o que
   queremos. Plano viavel sem hack.
